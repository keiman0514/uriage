# 店舗売上ダッシュボード

営業日報 Excel と本社の営業報告書 PDF をブラウザにドラッグ＆ドロップして、月別・年別・曜日別・期間別に比較するための静的Webアプリです。

## 使い方

1. GitHub PagesのURLを開きます。ローカルで試す場合は、このフォルダを小さなローカルサーバーで開きます。
2. 営業日報の `.xlsm` / `.xlsx` と、営業報告書の `.pdf` をドロップします。
3. 月別推移、曜日別ランチ比較、期間比較、黒字/赤字判定を確認します。
4. キャンペーンや出来事を登録すると、期間中と直前期間の比較が出ます。
5. 定期的に「バックアップ保存」で集計済みデータを書き出します。

## 読み取りルール

- Excelの年月はファイル名から読み取ります。例: `西荻営業日報2025年4月.xlsm`
- PDFの年月はファイル名から読み取ります。例: `1008 202504 営業報告書.pdf`
- Excelは `入力` シートから日別データを読み取ります。
- PDFは売上、原価、経費、人件費、利益などを読み取ります。
- 祝日は日本の祝日ルールで自動判定します。

## 保存について

読み取った集計データは、そのPCのブラウザ内に保存されます。Excel/PDFファイルそのものは保存しません。

GitHub Pagesで公開して使う場合も、ファイルの中身はブラウザ内で処理されます。別PCへ移すときは「バックアップ保存」と「バックアップ読込」を使ってください。

## GitHub Pagesに置く場合

1. このフォルダの中身をGitHubリポジトリに入れます。
2. GitHubの `Settings` から `Pages` を開きます。
3. `Deploy from a branch` を選び、`main` ブランチのルートを公開します。
4. 発行されたURLをお店のPCで開きます。
